Research Management System: architecture, workflows, JUREC ethics certificate, thesis supervision, and roles.

1. Architecture

React frontend, Express API, MongoDB. Users pick UG or PG portal; data is separated by programTier.

flowchart TB users["4 roles (Browser)"] react["React + Vite UI"] api["Express /api/* + JWT"] db["MongoDB"] users --> react --> api --> db

Frontend

Pages: Ethics, Proposals, Projects, Grants, Budgets, Publications, Repository, Groups, Thesis, Workflow

Auth: sessionStorage per tab (dual-user safe)

Backend

REST controllers + RBAC middleware

JUREC PDF, notifications, project-scoped data, OAI-PMH export
