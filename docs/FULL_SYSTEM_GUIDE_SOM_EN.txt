1. WHAT IS THIS SYSTEM? / WAA MAXAY SYSTEMKAN?
================================================================================

English: A web app for managing university research from funding call to project closure.

Somali: App web ah oo lagu maamulo cilmi-baarista jaamacadda — laga bilaabo funding call ilaa xiritaanka mashruuca.

English: Two portals — Undergraduate (UG) and Postgraduate (PG). Data is separated.

Somali: Laba portal — UG iyo PG. Xogtu waa kala go'an.

English: Only Research Director can switch between UG and PG.

Somali: Kaliya Research Director ayaa beddeli kara UG iyo PG.

2. WHO USES IT? / YAA ISTICMAALA? — SHAQADA MID KASTA (8 ROLES)
================================================================================

There is no separate Ethics Committee or Peer Reviewer account.
Ma jiro akoon gooni ah oo Ethics Committee ama Peer Reviewer.

Research Director
  EN: Portal switch; funding calls; final proposal approve; ethics decide; PO after Procurement; Users/Departments; analytics
  SO: Portal; funding calls; ansixinta ugu dambeysa; ethics; PO ka dib Procurement; Users; analytics

University Leadership
  EN: Publish funding calls; peer review score 1–5 + comment; KPI
  SO: Daabaca funding calls; peer review; KPI

Faculty Coordinator
  EN: Faculty proposal queue; thesis groups; Validate publications → Published; faculty reports
  SO: Proposal faculty; thesis; Validate → Published; warbixin faculty

Finance Officer — MONEY (not Procurement)
  EN: Authorize grant budget; pay budgets/payments; pay PO after Director; finance reports; finance closure
  SO: Authorize budget; bixi lacag; bixi PO ka dib Director; reports; xiritaan maaliyadeed
  EN: Does NOT review what to buy
  SO: MA sameeyo dib u eegista iibsashada (Procurement)

Procurement Officer — BUYING / PO (not Finance)
  EN: Review/approve Purchase Orders before Director; does not authorize grants or pay
  SO: Dib u eegis PO ka hor Director; ma authorize grant; ma bixin lacag
  EN: Flow: Researcher PO → Procurement → Director → Finance pays
  SO: Safka: Researcher PO → Procurement → Director → Finance bixisaa

HR Officer
  EN: Projects, thesis, groups — HR view only (no money/ethics approvals)
  SO: Projects, thesis, groups — HR (ma ansixiyo lacag/ethics)

Donor Agency
  EN: Draft external funding calls; donor reports
  SO: Funding calls dibadda (draft); donor reports

Researcher
  EN: Apply, propose, run project, budgets, create PO, publications, thesis, closure
  SO: Codso, proposal, project, budgets, abuuri PO, publications, thesis, closure

4. FULL JOURNEY — STEP BY STEP / SAFKA BUUXA
================================================================================

STEP 1 — Funding & Calls / Deeqaha

EN: Director (or Donor draft + Leadership publish) opens Funding Calls. Researchers apply.
SO: Director / Leadership waxay daabacaan Funding Calls. Researchers waxay codsadaan.
Menu: Funding Calls, Grants

STEP 2 — Application (Proposal) / Codsiga

EN — Researcher:
  1. Proposals → New proposal
  2. Fill title, abstract, department, research area
  3. Upload main document
  4. Add budget lines
  5. Upload compliance + supporting docs
  6. Complete Ethics form (JUREC) on the same flow
  7. Save draft → Submit

SO — Researcher: Proposals → cusub → buuxi dhammaan → Submit
Menu: Proposals

STEP 3 — Review & Approval / Dib u eegis & Ansixin

EN order:
  1. Admin screening — Director/Coordinator
  2. Peer review — University Leadership (score + comment)
  3. Committee review
  4. Ethics — Research Director (not a separate ethics role)
  5. Finance review — Finance Officer (proposal budget)
  6. Final decision — Director Approve or Reject
  7. If approved → Project created; funding-call grant → pending_finance

SO: Admin → Leadership peer review → Committee → Ethics (Director) → Finance → Director Approve → Project
Menu: Proposals, Peer Reviews, Ethics, Grant funding approval

STEP 4 — Project Management / Maamulka Mashruuca

EN: Projects → timeline, milestones, team, work plan, activities, communication log, progress reports
SO: Projects → milestones, work plan, hawlaha, communication log, warbixinno
Menu: Projects, Finance & Budgets

STEP 5 — Finance & Procurement / Maaliyad & Iibsashada

EN:
  1. Finance authorizes grant budget (Grant funding approval) — allocate only
  2. Researcher uses Budget / payment requests
  3. Purchase orders: Researcher → Procurement → Director → Finance pays

SO:
  1. Finance Authorize budget
  2. Budget / lacag bixin
  3. PO: Researcher → Procurement → Director → Finance

Menu: Grant funding approval, Finance & Budgets, Finance Reports

STEP 6 — Outputs / Natiijooyinka

EN: Publications (1:1 per project). Create & submit → Coordinator/Director Validate = Published. Research Workflow stages, Repository, Thesis (min 4 students), Groups.
SO: Publications → Validate = Published. Repository, Thesis (min 4 arday), Groups.

STEP 7 — Monitoring & Reports / La Socodka

EN: Dashboard, KPI, Finance Reports, Donor Reports, Search, Audit Trail
SO: Dashboard, KPI, Finance Reports, Donor Reports, Search, Audit Trail

STEP 8 — Project Closure / Xiritaanka Mashruuca

EN — Researcher submits checklist + final report + lessons learned → Director → Finance → Archive / Closed
SO: Checklist + final report → Director → Finance → Archive
Menu: Projects → project → Project closure; Finance → Project closure (Finance)

6. SIMPLE PICTURE / SAWIR FUDUD
================================================================================

Funding Call → Grant Application
        ↓
Proposal + Ethics + Budget + Docs
        ↓
Multi-stage Review (Leadership peer review · Director ethics)
        ↓
Director Approves → Project Created · Grant pending_finance
        ↓
Finance Authorize budget → Work Plan + Activities + Budget
        ↓
Payments + Procurement
        ↓
Publications (Validate = Published) + Repository + Progress
        ↓
Closure Checklist + Lessons Learned
        ↓
Director + Finance Approve → Archived / Closed

7. ONE SENTENCE SUMMARY / HAL JUMLAD
================================================================================

English: You apply through a funding call with ethics and budget → staff review it → if approved you get a project → Finance authorizes budget → you run the project → Validate publications to Published → close with a checklist.

Somali: Waxaad ka codsataa funding call + ethics + budget → shaqaalaha way dib u eegaan → haddii la aqbalo waxaad heshaa project → Finance authorize → waad maamushaa → Validate = Published → xiritaan checklist.
